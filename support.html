Common issues

  • “Needs access to your Google Drive” – Click “Allow” on the Google consent screen; the extension only reads file metadata.
  • Empty CSV – Ensure the folder contains files you can access and you pasted the correct folder URL.
  • Org policy blocks OAuth – Ask your Google Workspace admin to allow third-party Drive API access for your account.
